## コンセプトワード

> **「風が記憶する、思いをつなぐ。」**

目には見えない風が、人と自然・人と人の間を行き来し、
思いを運び、記憶を刻む——そんな詩的な世界観をビジュアルで表現する。

| 軸 | キーワード |
|----|-----------|
| **感覚** | 清涼感・空気感・流れる・抜けていく |
| **感情** | つながり・思い・温かさ・静けさの中の躍動 |
| **視覚** | 開放的・余白・光・群青の深さ |

## カラーパレット

```
────────────────────────────────────────────────
  空青（そらあお）   #6EC6F0   爽やかな夏空・風の通り道
  群青（ぐんじょう） #1B3F8B   深さ・夜空・思いの重なり
  白濁（はくだく）   #F0F6FF   霞・光の拡散・余白の空気
  墨（すみ）        #1A1F2E   文字・静寂・大地
  金風（きんぷう）   #D4A84B   アクセント・祭りの輝き・熱量
────────────────────────────────────────────────
```

- **ベース**: 白濁（#F0F6FF）— 光が拡散した空気感
- **メイン**: 空青（#6EC6F0）— 見た瞬間に風を感じる色
- **ディープ**: 群青（#1B3F8B）— ヘッダー・フッター・重みが必要な場所
- **テキスト**: 墨（#1A1F2E）— 読みやすく、深みがある
- **アクセント**: 金風（#D4A84B）— よさこい鳴子の色・祭りのエネルギー（最小限に）

## ビジュアルモチーフ

### 風の表現
- **流線**: 薄い曲線をページ背景やセクション区切りに用いる（風の軌跡）
- **グラデーション**: 空青→群青の縦グラデーション（空の深さ）
- **余白**: テキストや写真を「風が吹き抜けるスペース」として積極的に活用

### 動きの表現（CSS Animation）
- ヒーロー画像上に薄い流線がゆっくり流れる（`animation: drift`）
- スクロールに合わせてコンテンツがふわりとフェードイン
- 過度なアニメーションは使わず、「感じるか感じないか」程度の繊細さ

### よさこいらしさ
- 躍動感のある写真をフルブリードで使用
- 鳴子の金色アクセントを要所に配置
- 集合写真は「人のつながり」が見えるトリミングを意識

- 写真に群青〜透明のグラデーションオーバーレイ
- キャッチコピーを白抜き文字で重ねる

### 記事一覧（マガジン型グリッド）
```
┌──────────────┐  ┌──────┐ ┌──────┐
│              │  │      │ │      │
│  大カード     │  │ 小   │ │ 小   │
│  (メイン記事) │  │      │ │      │
│              │  └──────┘ └──────┘
└──────────────┘  ┌──────┐ ┌──────┐
                  │      │ │      │
                  │ 小   │ │ 小   │
                  │      │ │      │
                  └──────┘ └──────┘
```
- 最新記事を大カードで強調
- カードは写真 + カテゴリタグ + タイトル + 日付のみ（本文抜粋なし）

### 記事ページ
- 本文幅を 720px に絞り、左右に余白（風の通り道）
- 読み進めると右側にふわりと目次が現れる
- コメント欄は本文と明確に分離し、穏やかな背景色（#E8F4FD）で囲む

## サイト用語（コピー・ナビ）

- **作品**: 年度ごとのよさこいの楽曲・振付・紹介の単位（ディスコグラフィ `/discography/`）。ナビ・見出し・`<title>` のセグメントに使う。
- **演舞**: 依頼・出演・実績など、会場で踊る「実演」の文脈に限定する（例: 演舞依頼ページ）。
- **SEO の詳細**: `<title>` / `description` のパターンは `docs/design.md` の「SEO・表記・メタ情報」を参照。
- **XML サイトマップ**: ビルド時に `@astrojs/sitemap` で `sitemap-index.xml` 等を生成。`public/robots.txt` で sitemap URL を宣言（詳細は `docs/design.md`）。

### グローバルナビ・フッター（導線）

- **ヘッダー**: ロゴ、**トップ**、**作品**（`/discography/`）、**演舞依頼**（`/enbu-irai/`）、**スケジュール**、**メンバー募集**、**ブログ**（`/articles/`）、**お問合せ**、**検索**。
- **フッター**: 上記と同じ主要リンクを「サイトマップ」として列挙し、あわせて SNS リンクを配置。
